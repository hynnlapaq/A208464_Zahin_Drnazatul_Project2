## About
JobHub is an Android job platform connecting Malaysian B40 community members to employment opportunities.

## Features
- 8 screens with Jetpack Compose Navigation
- Room Database (local persistence)
- Firebase Firestore (cloud sync)
- REST Countries API (live Malaysia data)
- Camera sensor — IC card OCR scanning (ML Kit)
- GPS sensor — Nearby Jobs (FusedLocation API)

## Setup
1. Clone this repo and extract the ZIP
2. Open in Android Studio
3. Add your google-services.json from Firebase Console
4. Run on emulator or physical device (API 26+)
